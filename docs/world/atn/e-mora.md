# 数字摩拉

## 简介

`atnttp://e-mora.org`

数字摩拉是由联合提瓦特、璃月黄金屋、璃月央行·人民银行联合发行的数字货币。

联合提瓦特负责数字摩拉在提瓦特跨国网络上的流通。
璃月黄金屋，璃月人民银行作为提瓦特唯一铸币机构和璃月央行负责负责数字摩拉的发行和，并配合各国进行监管。
