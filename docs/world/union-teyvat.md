# 联合提瓦特 Union Teyvat

## 简介

联合提瓦特是一个由多个提瓦特国家组成的组织。它的成立是为了促进提瓦特社会之间的合作和合作。

总部地址：璃月·云来海·璃月港·玉京台·联合提瓦特总部
