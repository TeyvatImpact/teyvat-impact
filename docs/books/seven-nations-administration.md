# 枫丹 - Fontaine

「水与正义」

枫丹是前「尘世七执政·水」芙卡洛斯的的国度，信仰「正义」。

当代枫丹的日常治理由最高审判官那维莱特主导，而神明芙宁娜（目前已经卸下神位）在长久以来更多地作为象征性的存在。

## 核心架构：神明与审判官

- 神明（已卸下神位）：芙宁娜
- 最高审判官：那维莱特
  枫丹目前的最高行政长官。办公地点位于沫芒宫一楼。虽然名义上主要负责审判，但枫丹几乎所有重要的行政项目、预算审批最终都需要经过他的过目与批准。

## 七大职能部门

枫丹的行政事务由七大主要机构分担。

### 逐影庭
直属于最高审判官那维莱特。负责重大、机密或特殊案件的调查工作，可视为那维莱特的直属亲信与特种调查部队。因其工作需要能看见「人类无法察觉的线索」的能力，成员多为具有此种天赋的美露莘。

### 执律庭
枫丹的公共安全与治安维护核心机构，相当于警察部门。
下有：
  - 警备机关：负责日常的安全巡逻。
  - 警备队：处理普通的案件。
  - 特巡队：执律庭中的精锐力量，负责处理特别危险、复杂的案件，如犯人拒捕、恶性犯罪等。拥有铳枪作为武器。队长：夏沃蕾。

### 审判庭
枫丹司法体系的核心，审判活动进行的场所与机构总称。主要工作地点为欧庇克莱歌剧院。下有检律庭，负责具体的审判工作。由于那维莱特精力有限，除重大案件外，大多数日常审判由检律庭负责，并依赖「谕示裁定枢机」（4.2 版本之前）做出最终判决。

### 复律庭
负责枫丹的综合民生事务，涵盖范围极广。大至国家级基建项目，小至民间商业活动的经营许可审批。工作地点为沐芒宫。~~底层职员加班现象较为普遍，官僚主义严重。~~

### 枢律庭
枫丹的财政审批部门，相当于财政部。负责统计、审核并发放枫丹各行政机关运行所需的资金，处理各部门的报销申请。巨额款项需上报那维莱特亲自批示。~~工作相对枯燥，且月底因集中处理报销而工作量激增。~~
